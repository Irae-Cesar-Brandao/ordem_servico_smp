Sistema para uso educacional, sem fins lucrativos, representação de um sistema de ORDEM DE SEVIÇO

Escopo: Estrutura do Sistema
O sistema ORDEM DE SERVIÇO / ATENDIMENTOS terá:

Cadastro do Cliente
- Nome
- Telefone
- Endereço
- Cidade
- Estado

Dados do Atendimento

- Tipo de equipamento
- Tipo de atendimento
- Remoto
- Presencial
- Laboratório
- Descrição do problema
- Execução

Data
Hora início
Hora término
Descrição da execução
Valor do serviço
Evidências
- Foto inicial (registro da entrada)
- Foto final (registro da conclusão)

Validação
- Validação via WhatsApp
- Confirmar início
- Confirmar finalização

Exportação :  CSV
